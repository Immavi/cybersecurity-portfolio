# My Cybersecurity Portfolio Strategy

## Goal
Document my 10-month journey from student to Cloud Security Engineer.

---

## Portfolio Platforms

| Platform | Purpose |
|---|---|
| **GitHub** | Main portfolio — notes, scripts, projects |
| **LinkedIn** | Share progress posts, connect with industry |
| **Medium/Hashnode** | Write articles about what I learn |

---

## What to Document (Everything!)

### For Each Certification
- Study notes (Markdown files)
- Key concepts in your own words
- Labs with screenshots
- Practice exam scores
- Final exam result

### For Each Module (University)
- Assignments completed
- Projects built
- Key things learned

### For Each Lab/Project
- What the lab was about
- Steps you followed
- What you learned
- Screenshots
